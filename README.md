## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/hk2166/Blynk_project.git
   cd Blynk_project
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. **Setup Backend**
   ```bash
   cd backend
   npm install
   npm start
   ```
